#DYNAMIC-MERGER-MODEL-DASHBOARD

Developed a dynamic merger financial model in Python that automatically generates an interactive Excel dashboard using XlsxWriter. The model integrates acquirer and target financials, deal terms, and synergy analysis to calculate pro forma metrics and EPS accretion/dilution, providing quick and clear deal evaluation.

•	Implemented conditional formatting to visually highlight accretive or dilutive outcomes for effective decision-making.

•	Enabled flexible scenario analysis with customizable inputs for purchase price, payment mix, synergies, and financing structure.
